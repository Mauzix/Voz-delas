# Documentação do Projeto — Acolher

## Sobre o projeto

O projeto **Acolher** é uma plataforma web informativa desenvolvida com foco em conscientização, orientação e apoio para mulheres em situação de violência doméstica. O site apresenta informações sobre os diferentes tipos de violência, canais oficiais de denúncia, medidas de proteção e conteúdos de acolhimento.

A proposta do sistema é oferecer uma experiência acessível, clara e humanizada, priorizando segurança, legibilidade e navegação intuitiva.

---

# Objetivos do projeto

* Informar sobre os diferentes tipos de violência doméstica.
* Facilitar o acesso aos canais oficiais de ajuda.
* Promover conscientização e acolhimento.
* Criar uma interface moderna, responsiva e acessível.
* Aplicar boas práticas de HTML, CSS e JavaScript.

---

# Tecnologias utilizadas

## HTML5

Utilizado para estruturar semanticamente todo o conteúdo da página, organizando seções, navegação, textos, botões e elementos de acessibilidade.

## CSS3

Responsável pela estilização da interface, incluindo:

* Variáveis CSS
* Responsividade
* Tema claro e escuro
* Animações suaves
* Grid Layout
* Flexbox
* Glassmorphism
* Sistema de espaçamento e tipografia

## JavaScript

Usado para adicionar interatividade e melhorar a experiência do usuário:

* Alternância entre tema claro/escuro
* Barra de progresso da página
* Sistema de toast/notificações
* Animações com Intersection Observer
* Botão de saída rápida
* Eventos de interação

# Estrutura da página

## Header

Contém:

* Logo da plataforma
* Navegação principal
* Botão de alternância de tema

## Hero Section

Área inicial de destaque com:

* Mensagem principal de conscientização
* Botões de ação rápida
* Informações sobre os canais 180 e 190

## Tipos de Violência

Apresenta explicações sobre:

* Violência física
* Violência psicológica
* Violência sexual
* Violência patrimonial e moral

## Como agir

Seção com orientações básicas de segurança e planejamento em situações de risco.

## Rede de apoio

Explica direitos, acolhimento e acesso aos serviços especializados.

## Emergência

Área de destaque com acesso rápido aos números:

* 180
* 190

## FAQ

Perguntas frequentes relacionadas à violência doméstica e canais de ajuda.

## Footer

Contém informações finais e lembretes importantes sobre segurança.

O projeto inclui práticas de acessibilidade para melhorar a navegação:

* Skip link

O layout foi desenvolvido utilizando:

* Media queries
* CSS Grid

* Smartphones
* Tablets
* Notebooks
* Monitores maiores

# Funcionalidades principais

## Alternância de tema

Permite trocar entre modo claro e escuro dinamicamente.

## Barra de progresso

Indica visualmente o progresso da rolagem da página.

## Toasts informativos

Exibem mensagens rápidas de orientação ao usuário.

Anima elementos conforme aparecem na tela.

# Considerações finais

O projeto busca unir tecnologia, acessibilidade e conscientização social em uma interface moderna e funcional. Além do aspecto visual, a proposta prioriza acolhimento, clareza das informações e acesso rápido à ajuda especializada.
